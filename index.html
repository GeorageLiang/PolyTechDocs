jekyll安装步骤(windows环境下请参看这里).

快速安装指南

~ $ gem install jekyll

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ jekyll serve

# => 现在可以通过浏览器访问 http://localhost:4000

GitHub Pages提供的免费服务器支持

GitHub Pages提供了该项目的网站托管支持，任何人都可以轻松将自己的基于Jekyll引擎的网站发布到GitHub上，该网站采用了Jekyll主页的样式和模板。
